# 短链接生成器
基于 Cloudflare Workers + D1 的轻量级短链接服务，支持密码保护、自定义有效期、多域名、智能复用等功能。

## 功能特性
- 短链接生成 – 将长网址压缩为简洁的 6 位短码。
- 密码保护 – 可设置访问密码，验证后 1 小时内无需重复输入。
- 自定义有效期 – 支持永久有效、1/7/30 天或指定过期日期，过期后自动清理。
- 多域名支持 – 可配置多个自有域名，前端下拉选择生成对应域名的短链。
- 智能复用 – 同一域名下若已存在未过期的相同原网址，直接返回已有短链，避免冗余。
- 定时清理 – 每天凌晨自动删除过期的数据库记录，保持轻量。
